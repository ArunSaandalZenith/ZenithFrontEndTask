# Zenith Angular Front End

## Developer Guidelines

Please complete the tasks you have been assigned bearing in mind the following:

- Try to adhere to [Angular Best Practices](https://angular.dev/style-guide)
- Use signals as a minimum for UI properties, but use whatever approach you are comfortable with
- Follow the smart / dumb pattern to keep components light, testable and maintainable
- Cover any code you create with unit tests, coverage should sit at a minimum of 80%
